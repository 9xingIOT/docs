# 九星云平台简介

为了打造**一站式便捷使用**的全终端应用，九星物联网自研九星云平台，包括**web 端（桌面网页）、公众号、微信小程序、手机 APP（内测中）、上位机（九星配置工具）**等一系列应用，为智能设备联网通信提供全方位保障。
